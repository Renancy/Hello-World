All above were run at terminal

  • git clone # clones a especific repository to your machine
  • git touch x.file #create an empty file
  • git add x.file #add a file I want to track next time I save
  • git commit -m "message"
  • git status #tell the current status of your repository
  • git push # send back the changes you made
  • git pull # the inverse of git push
  • git log # show all the commits the file has passed by
  • git reset --hard #reset to a specific commited older version
  • git commit -am #commit and add to the staging area, simplifies the code
